<h1>Access-Based-Enumeration</h1>
<p>
  Access-Based-Enumeration is a feature in Windows server that hides files and fodlers from users who do not have permission to access.
  Enhance security prevents users knowing certain folders if don't have permission, reduces users confusion with large share drive and clean showing releveant folders,
  and protects privacy of sensitive folders, enforce privacy policies and data protection regulations, and limits unnecessary access attempts.
</p>

<h2>Real-World Simulation:</h2>
<p>I am simulating a file server for the HR and IT departments. I want users from HR to only see HR-related folders and IT to only see theirs</p>
